# Counter App
Una volta aperta, l'applicazione si presenta con un numero 0 e tre bottoni sotto di esso:<br>
il tasto - decrementa il numero di un'unità<br>
il tasto "clear" azzera il numero<br>
il tasto + aumenta di un'unità.
